    • Mixedmodels-misc

    miscellaneous materials for mixed models, mostly in R

    Project maintained by bbolker Hosted on GitHub Pages — Theme by mattgraham

    This repository provides a variety of resources I've written, or compiled, about mixed models; the focus is on the tools I help develop and maintain (the lme4, glmmTMB, and glmmADMB packages for R). (See the Github repo for source code.)
